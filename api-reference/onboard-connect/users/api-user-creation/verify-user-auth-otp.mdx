---
title: 'Verify User Authentication OTP'
openapi: post /auth/user-auth/verify-otp
---

Verifies the OTP provided by a user for authentication during sign-in or sign-up.

<Warning>
This endpoint is specifically for user authentication (logging in or signing up).

Do not confuse it with [`/users/me/send-otp`](generate-otp-for-2fa-actions), which generates OTPs for verifying sensitive account actions (e.g., updating payment methods).
</Warning>
