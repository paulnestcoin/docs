This section of the docs covers Onboard Exchange, a marketplace that matches buyers and sellers to swap crypto assets to various fiat currencies and vice-versa.

The Onboard Exchange dApp is currently a hybrid system comprising both onchain (the protocol itself) and off-chain components (for functionalities like dispute resolution and user onboarding).
